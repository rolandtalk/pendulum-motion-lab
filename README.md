# Pendulum Motion Lab

Interactive browser simulation for comparing two Lagrangian cart-and-pendulum systems:

- a chain setup where `m2` hangs below `m1`
- a direct setup where both `m1` and `m2` connect directly to the cart

The app is static and deploys without a build step. Use the project folder itself as the publish directory.

Suggested Cloudflare Pages project name:

```text
pendulum-motion-lab
```

Suggested production URL:

```text
https://pendulum-motion-lab.pages.dev
```
